# 二叉树遍历所有路径的模板

> 注意：和图的遍历是有些不同的

```C++
vector<vector<int>> ans;
void traversal(vector<int>& path, TreeNode* root) {
	path.push_back(root->val);
	// 完成一条路径
	if (!root->left && !root->right) {
		ans.push_back(path);
		return;
	}
	if (root->left) {
		traversal(path, root->left);
		path.pop_back();
	}
	if (root->right) {
		traversal(path, root->right);
		path.pop_back();
	}
}
```
